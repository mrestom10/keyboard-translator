## Full Description (4000 chars max)
Translator Keyboard is a powerful keyboard app that lets you type and translate text in real-time, directly from any app on your phone. No need to switch between apps — just type, translate, and send.

TRANSLATE AS YOU TYPE
Type your message in any app (WhatsApp, Telegram, Instagram, Email, etc.), tap the Translate button, and instantly get the translation. Replace your original text with the translated version or send it directly — all without leaving the conversation.

SUPPORTED LANGUAGES
- English
- Arabic (full Arabic keyboard layout included)
- French
- German
More languages coming soon!

FULL CUSTOM KEYBOARD
- Complete QWERTY keyboard with long-press for accented characters (e.g., hold 'a' for àáâäæãåā)
- Full Arabic keyboard with all 28 letters + diacritics on long-press
- Number row (always visible, toggleable)
- Numbers, symbols, and special characters keyboard
- Haptic feedback on every keypress
- Shift, backspace with long-press repeat, enter key

GIF & STICKERS
- Search and send GIFs powered by GIPHY
- Browse trending GIFs and stickers
- Animated sticker search
- Tap to insert into any chat

EMOJI PICKER
- 6 emoji categories: Smileys, Love, Animals, Food, Sports, Travel
- Full Unicode emoji support
- Quick access from the toolbar

PRIVACY FOCUSED
- Your keystrokes are NEVER logged or stored
- Translation only happens when YOU tap the button
- No analytics, no tracking, no data collection
- All processing happens on your device

HOW TO SET UP
1. Install the app
2. Open Settings > System > Languages & Input > On-screen keyboard
3. Enable "Translator Keyboard"
4. Open any app and switch to Translator Keyboard

PERFECT FOR
- Travelers communicating in foreign languages
- Multilingual conversations
- Learning new languages
- International business communication
- Chatting with friends and family who speak different languages

This keyboard respects your privacy. We only send text to translation services when you explicitly tap Translate. No background data collection, no keystroke logging, ever.
